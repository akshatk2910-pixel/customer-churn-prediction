# Customer Churn Prediction

## Overview
This project predicts customer churn using Logistic Regression implemented from scratch using NumPy.

## Dataset
Telco Customer Churn Dataset

## Features
- Data Cleaning
- Missing Value Handling
- Binary Encoding
- One-Hot Encoding
- Feature Scaling
- Logistic Regression from Scratch
- Gradient Descent
- Threshold Tuning

## Results
- Accuracy: ~80%
- Precision: 60%
- Recall: 68%
- F1 Score: 63.6%

## Technologies Used
- Python
- NumPy
- Pandas
- Jupyter Notebook
